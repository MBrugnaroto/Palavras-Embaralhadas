## Palavras-Embaralhadas
### Descubra a palavra

#### Objetivo do projeto
<p> Desenvolver uma aplicação onde se pudesse aplicar técnicas da programação orientada a objetos.</p>

#### Introdução
<p> O jogo é simples: será apresentado para o jogador uma palavra que estará com as letras embaralhadas e o mesmo terá como objetivo adivinhar a palavra em questão. </p>

#### Informações gerais
<p> Como jogar: no menu principal selecione uma dos dois modos de jogo: fácil ou dificil. A partir desse ponto, será apresentado no display uma palavra embaralhada e um campo onde você poderá inserir a palavra que voce deduziu ser. Tanto o modo fácil quanto o modo difícil você terá apenas 3 chances de acertar a palavra e um total de 5 palavras embaralhadas. Boa sorte!!

<p> Navegação: para voltar para o menu principal ou fazer algum outro acesso, siga as instruções disponiveis no display. Caso queira sair no meio de uma partida digite EXIT com todas as letras em maiúsculo.</p>

#### Modos de Jogo
| Modo | Informação | Modelo de Pontuação |
| ---- |:----------:|:-------------------:|
|`Fácil`| A palavra sofre um embaralhamento e dentro das 3 tentativas você a verá do mesmo modo, ou seja, no mesmo formado da primeira tentativa. | <p> Acerto na primeira tentativa: 15 pontos</p><p> Acerto na segunda tentativa : 10 pontos</p><p> Acerto na terceira tentativa: 5  pontos</p><p> Errou a palavra             : 0  pontos</p> |
|`Difícil| A palavras sofre um embaralhamento e dentro das 3 tentativas você a verá em formatos diferentes, ou seja, a palavra será mostra em formatos diferentes. | <p> Acerto na primeira tentativa: 30 pontos</p><p> Acerto na segunda tentativa : 20 pontos</p><p> Acerto na terceira tentativa: 10 pontos</p><p> Errou a palavra             : 0  pontos</p> |
